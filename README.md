#Especialidade de Programação: Bloco de especialidades
Link do site já hospedade: https://bloco-de-especialidades.netlify.app/

Nesse projeto você pode cadastrar as especialidades que você que tirar no escotismo e armazenar no localStorge simulando um banco de dados.

No index você digita o nome da especialidade e seleciona, automaticamente a imagem correspondente aparece.
